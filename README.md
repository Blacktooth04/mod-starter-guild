# Starting Guild #

### Description ###
------------------------------------------------------------------------------------------------------------------
This module auto-assigns new players to specific guilds.

- New characters are auto-joined to a specified guild on first login only.
- Config: Set ID for Horde and Alliance guilds to use.
	
### Data ###
------------------------------------------------------------------------------------------------------------------
- Type: Player/Server
- Script: StartGuild
- Config: Yes
